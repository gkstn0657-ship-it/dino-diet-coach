## 2. 화면 이동 흐름도

```mermaid
graph TD
    Landing[랜딩] --> Login[로그인]
    Landing --> Regist[회원가입]
    Regist --> Login
    Login --> Home[홈]

    Home --> DietList[식단 목록]
    Home --> Coach[AI 운동 코칭]
    Home --> ChList[챌린지 목록]
    Home --> Board[커뮤니티]
    Home --> MyPage[마이페이지]

    DietList --> DietWrite[식단 기록<br/>사진→AI 비전 분석]
    DietList --> DietDetail[식단 상세<br/>영양 분석]
    DietDetail --> DietModify[식단 수정]
    DietDetail --> AiAnalysis[AI 식단 종합 분석]

    Coach --> CoachSelect{코치 선택}
    CoachSelect --> Rex[PowerRex 🦖<br/>벌크업]
    CoachSelect --> Dino[SlimDino 🦕<br/>다이어트]
    CoachSelect --> Turtle[BalanceNo 🐢<br/>유지]

    ChList --> ChCreate[챌린지 생성]
    ChList --> ChDetail[챌린지 상세<br/>참여·일일 인증]
    ChDetail --> MyCh[내 챌린지 현황]

    Board --> PostWrite[글쓰기]
    Board --> PostDetail[게시글 상세<br/>댓글]
    PostDetail --> PostModify[글 수정]
    PostDetail --> UserProfile[사용자 프로필<br/>팔로우]

    MyPage --> MemberModify[프로필 수정<br/>회원 탈퇴]
```

## 3. 주요 화면 레이아웃

### 3-1. 식단 기록 (F101 + F116심화)

```
┌────────────────────────────────┐
│  식단 기록하기 📸               │
│  ┌──────────────────────────┐  │
│  │   📷 사진 업로드 영역      │  │
│  └──────────────────────────┘  │
│  [ 🦕 AI 영양 분석 ]  ← 비전 AI │
│  ┌ 분석 결과 (자동 채움) ────┐  │
│  │ 음식명·칼로리·영양소       │  │
│  └──────────────────────────┘  │
│  음식 DB 검색·선택 / 끼니·날짜  │
│  [ 저장 ]                      │
└────────────────────────────────┘
```

### 3-2. AI 운동 코칭 (F117)

```
┌────────────────────────────────┐
│  AI 운동 코칭 💪                │
│ ┌────────┬────────┬─────────┐ │
│ │PowerRex│SlimDino│BalanceNo│ │  ← 코치 선택 카드
│ │  🦖    │  🦕    │   🐢    │ │
│ └────────┴────────┴─────────┘ │
│ ┌──────────────────────────┐  │
│ │ 🦖 PowerRex (헤더)        │  │
│ │ ┌─ 채팅 영역 ───────────┐ │  │
│ │ │ 코치: 인사/저장된 대화 │ │  │
│ │ │           사용자: 질문 │ │  │
│ │ │ 코치: 운동·식단·동기   │ │  │
│ │ └───────────────────────┘ │  │
│ │ [입력창]          [전송]  │  │
│ └──────────────────────────┘  │
└────────────────────────────────┘
```
