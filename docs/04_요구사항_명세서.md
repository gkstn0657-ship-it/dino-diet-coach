## 4. 비기능 요구사항

| 구분 | 내용 |
|------|------|
| 보안 | 비밀번호 해시 저장, JWT 만료(30분), 인증 인터셉터(`/api/v1/**`), AI 히스토리 role 화이트리스트로 프롬프트 주입 차단, 코치 프롬프트 물결표 금지로 마크다운 렌더링 오류 방지 |
| 가용성 | AI 키 미설정/호출 실패 시 fallback 응답으로 서비스 지속, 대화 저장 실패가 응답을 막지 않음 |
| 성능 | LLM 전송 히스토리 10턴 제한(토큰 상한), 비전 이미지 512px 리사이즈, 조회 인덱스 설계 |
| 데이터 보존 | AI 대화 90일 보존 후 자동 삭제, 회원 탈퇴 시 연관 데이터 CASCADE |
| API 규약 | REST `/api/v1/**`, 공통 응답 래퍼(ApiResponse), 전역 예외 처리 |
